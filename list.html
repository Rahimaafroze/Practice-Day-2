Inline element

    Inline Element stays in-line. Inline elemnet doesn't starts a new line. Margin,Padding,Border can be added but can not accept width and height.Allow other html element to sit to their left and right.

  1. Anchor tag: a
  2. Span tag
  3. Image tag: img
  4. tag i
  5. Strong tag
  6. Small tag
  7. Iframe
  8. Audio-if it has visible controls

    9. Block Element

    Block element starts a new line. Only take border,margin,height,width,paddingAcquires extra space i.e acquires full width if width is not defined.

  9. Div tag
  10. Aside tag
  11. Article tag-used specially in blog(semantic)
  12. h1-h6 tags
  13. Footer
  14. Form
  15. Header
  16. hr tag
  17. Section
  18. Table
  19. ul tag

    20. Inline-Block element

    Inline-Blockelement is placed as an inline(in a line)but behaves like block element.Height and width can be set.Mostly used creating navigation bar.
